# Hibernate Student Management Assignment

## Description
A simple Java application demonstrating Hibernate ORM (without Spring Boot) for persisting Student data into MySQL. The project uses JPA annotations, SessionFactory, Transactions, and automatic schema generation.

## Requirements
- Java 23
- MySQL
- Maven

## Setup & Run
1. Clone the repository.
2. Create a MySQL database named `hibernate_db`.
3. Copy `hibernate.cfg.xml.template` to `hibernate.cfg.xml` in `src/main/resources/`.
4. Replace `${DB_USERNAME}` and `${DB_PASSWORD}` with your local MySQL credentials.
5. Run `Main.java` to persist sample Student data.
